### Config env variables

Location :  `(Create .env in root directory of project)`

```env
    POSTGRES_USER=your-database-user
    POSTGRES_PASSWORD=your-database-password
    POSTGRES_DB=your-database-name
    SECRET_KEY="DJANGO SECRET KEY"
    DEBUG=False
```
